## Project Highlights

- Built Assure AI, an AI tool that automates complaint case workflows from document upload to investigation outputs.
- Created document pipelines for ingestion, classification, parsing, routing, and folder management.
- Developed LLM workflows to generate SOC drafts, evidence summaries, chronologies, staff statements, and report sections.
- Implemented large-document handling with page chunking, structured extraction, JSON validation, repair prompts, and token tracking.
- Designed a reliable backend workflow with case metadata, document manifests, reusable prompts, and automated output storage.
