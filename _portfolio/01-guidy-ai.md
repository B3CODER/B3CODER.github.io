## Project Highlights

- Built an AI customer support agent for SIM queries, customer plans, purchases, and ticketing workflows.
- Engineered a RAG system to retrieve accurate knowledge-base answers and generate context-aware responses.
- Developed an end-to-end query routing pipeline using user intent, chat history, and page context.
- Implemented multi-pipeline handling for Q&A, commands, navigation, form filling, and support tasks.
- Integrated LLM intent classification, vector search, embeddings, reranking, multilingual support, and token tracking.
